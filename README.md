# 🤖 Study AI — The Cognitive Sanctuary

> A premium, local-first AI study suite powered by RAG, local LLMs, and the **Stitch AI "Lumina Academic"** design system.

Study AI transforms your PDFs into an interactive learning experience — chat with your documents, generate quizzes, build mind maps, create cheat sheets, plan your study week, and more. All powered by local LLMs through Ollama, so your data never leaves your machine.

---

## ✨ Feature Arsenal

| Feature | Description |
|---|---|
| 💬 **Intelligent RAG Chat** | Ask questions grounded in your uploaded PDFs with source attribution |
| 🎭 **Persona Explainer** | Switch AI teaching styles: ELI5, PhD Level, Analogy Mode |
| 📊 **Auto-Mermaid Diagrams** | AI automatically generates visual flowcharts for processes |
| 📝 **MCQ Quiz Generator** | AI-generated 5-question quizzes to test your knowledge |
| 📚 **Master Summary** | Structured "CliffNotes" study guide (TL;DR, key terms, takeaways) |
| 🗂️ **Flashcard Center** | Interactive flip-card active recall system |
| 🧠 **Mind Map Explorer** | Visual hierarchical concept maps via Mermaid mindmaps |
| 📊 **Cheat Sheet Generator** | Top 10 terms/formulas extracted into a quick-reference table |
| 📅 **AI Study Planner** | Auto-generated 5-day curriculum based on your material |
| ⏱️ **Pomodoro Timer** | Built-in 25/5 focus timer in the sidebar |
| 🔊 **Voice TTS** | Read Aloud + Stop buttons using browser Web Speech API |
| 📎 **Source Clipping** | View exact document excerpts used by the AI |
| 📤 **Export & Share** | Download chat sessions as JSON |

## 🎨 Design System: Lumina Academic

The UI is powered by design tokens generated by **Stitch AI**, following the *"Cognitive Sanctuary"* creative direction:

- **Typography**: `Space Grotesk` (headlines) + `Manrope` (body) + `JetBrains Mono` (code)

## 🛠️ Tech Stack

- **Frontend**: [Streamlit](https://streamlit.io/) with massive custom CSS injection
- **LLM Engine**: [Ollama](https://ollama.com/) (Local — `llama3`, `mistral`, `gemma`, `phi3`)
- **Vector Database**: [FAISS](https://github.com/facebookresearch/faiss)
- **Embeddings**: `sentence-transformers` (`all-MiniLM-L6-v2`)
- **PDF Parsing**: `pdfplumber`
- **Visuals**: `Mermaid.js` (custom HTML components)
- **Design**: [Stitch AI](https://stitch.google.com/) (Lumina Academic tokens)

## 🚀 Getting Started

### 1. Prerequisites
- **Python 3.10+**
- **Ollama** installed and running

### 2. Install Ollama & Pull Models
```bash
ollama pull llama3
ollama pull mistral
```

### 3. Clone & Install
```bash
git clone https://github.com/likhith3035/studyai.git
cd studyai

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt
```

### 4. Run
```bash
streamlit run app.py
```

## 📖 How to Use

1. **Upload**: Go to **🛠️ Admin Space** → Upload study PDFs.
2. **Chat**: Switch to **💬 User Chat** → Ask questions. Pick a Persona for different explanation styles.
3. **Study Tools**: Use **📝 Quiz**, **🗂️ Flashcards**, **🧠 Mind Map**, **📊 Cheat Sheet**, or **📅 Study Planner**.
4. **Focus**: Start the **⏱️ Pomodoro Timer** in the sidebar.
5. **Listen**: Click **🔊 Read Aloud** on any AI response. Click **🔇 Stop** to cancel.
6. **Export**: Download your chat as JSON from the sidebar.

- **Ollama must be running** locally at `http://localhost:11434`
- **Privacy**: Your data stays on your machine. Nothing is sent to external servers.
- **Resource Usage**: First load indexes all PDFs. Large files may take a few seconds.
